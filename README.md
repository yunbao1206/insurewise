# WiseClick Insure

> 8 free online insurance calculators. Estimate your coverage needs.

**Live site:** [insure.wiseclick.site](https://insure.wiseclick.site)

## Calculators

- Auto Insurance Calculator
- Life Insurance Calculator
- Home Insurance Calculator
- Health Insurance Calculator
- Pet Insurance Calculator
- Travel Insurance Calculator
- Insurance Needs Calculator
- Deductible vs Premium Calculator

## Features

- Based on national average rate data
- All calculations run in-browser — your data stays private
- Side-by-side deductible comparison tool
- JSON-LD structured data (WebApplication + FAQPage schemas)
- Mobile responsive

## Disclaimer

These calculators provide estimates for educational purposes only. Results are not insurance quotes. Consult a licensed insurance agent for actual quotes.

## Tech

- Pure HTML/CSS/JavaScript
- Hosted on GitHub Pages
- Zero dependencies
